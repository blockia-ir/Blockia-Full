# نحوه نصب
ابتدا اسکریپت را دانلود کنید و آن را در هاست خود آپلود کنید .
سپس فایل را استخراج کرده و یک دیتابیس بسازید .
بعد از ساخت دیتابیس آدرس محل نصب + / + installation را بزنید .
```bash
site.com/installation
```
